# Comment créer une boutique en ligne avec WooCommerce ? Guide complet

WooCommerce est l'extension e-commerce la plus populaire au monde, propulsant **28 % des boutiques en ligne** mondiales. Gratuite, flexible et puissante, elle transforme n'importe quel site WordPress en boutique professionnelle. Voici comment la configurer étape par étape.

## Qu'est-ce que WooCommerce ?

WooCommerce est une **extension (plugin) gratuite pour WordPress** qui ajoute toutes les fonctionnalités d'une boutique en ligne à votre site : catalogue produits, panier, paiement sécurisé, gestion des stocks, calcul des frais de livraison, etc.

Il fonctionne avec n'importe quel thème WordPress et s'étend via des centaines d'extensions spécialisées.

## Prérequis avant de commencer

Avant d'installer WooCommerce, vous avez besoin :
- D'un **hébergement web** (avec PHP 7.4+ et MySQL 5.6+)
- D'un **nom de domaine**
- De **WordPress installé** sur votre hébergement
- D'un **certificat SSL** (HTTPS) — obligatoire pour les paiements en ligne

## Étape 1 : Installer WooCommerce

1. Connectez-vous à votre tableau de bord WordPress
2. Allez dans **Extensions → Ajouter**
3. Recherchez "WooCommerce"
4. Cliquez sur **Installer** puis **Activer**
5. L'assistant de configuration WooCommerce se lance automatiquement

## Étape 2 : Configurer les paramètres de base

L'assistant vous guidera pour paramétrer :

### Informations de votre boutique
- Adresse (utilisée pour le calcul des taxes et de la livraison)
- Secteur d'activité
- Type de produits (physiques, numériques, ou les deux)

### Mode de paiement
WooCommerce inclut nativement :
- **PayPal** : simple à configurer
- **Virement bancaire** : pour les clients B2B
- **Paiement à la livraison**

Pour accepter les cartes bancaires, installez l'extension **Stripe** (gratuite, commission de 1,4 % + 0,25 € par transaction en Europe).

### Livraison
Configurez vos zones de livraison et tarifs :
- Livraison gratuite à partir d'un montant minimum
- Tarif fixe par commande
- Tarif calculé au poids ou à la destination

## Étape 3 : Choisir un thème compatible WooCommerce

Tous les thèmes WordPress ne sont pas optimisés pour WooCommerce. Choisissez parmi :
- **Astra** (gratuit, très rapide, recommandé)
- **Flatsome** (payant, populaire pour le e-commerce)
- **Divi** (payant, constructeur de page intégré)
- **Storefront** (gratuit, thème officiel WooCommerce)

## Étape 4 : Ajouter vos premiers produits

1. Dans l'administration, allez dans **Produits → Ajouter**
2. Renseignez :
   - **Nom du produit** (optimisé avec vos mots-clés)
   - **Description courte** (affichée dans les listings)
   - **Description longue** (affichée sur la fiche produit)
   - **Prix** (et prix promo si applicable)
   - **Stock** (gestion automatique si activée)
   - **Images** (principale + galerie)
   - **Catégorie et tags**

3. Cliquez sur **Publier**

## Étape 5 : Configurer les taxes

En France, vous devez appliquer la TVA sur vos ventes. WooCommerce gère automatiquement les taux selon les pays si vous :
1. Activez les taxes dans **WooCommerce → Réglages → Général**
2. Configurez les taux dans l'onglet **Taxes**

Taux français standards : 20 % (standard), 10 % (restauration), 5,5 % (alimentaire), 2,1 % (médicaments).

## Étape 6 : Optimiser votre boutique pour le SEO

Une boutique WooCommerce non optimisée pour le SEO est invisible sur Google. Installez **Yoast SEO** ou **Rank Math** et optimisez :

- **Titre et méta description** de chaque fiche produit (avec mots-clés acheteur)
- **URLs propres** : mindzy.me/produit/nom-produit/ plutôt que ?p=123
- **Balises alt** sur toutes les images produit
- **Données structurées produit** (prix, disponibilité) pour les rich snippets Google Shopping
- **Vitesse de chargement** : des images produit optimisées sont essentielles
