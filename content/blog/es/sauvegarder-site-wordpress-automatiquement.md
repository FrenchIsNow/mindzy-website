# Cómo hacer copias de seguridad automáticas de tu sitio WordPress

Tu sitio web es un activo de negocio. Como cualquier activo, necesita protección. Una copia de seguridad es tu póliza de seguro contra el escenario que todo propietario de web teme: despertar con el sitio hackeado, roto o eliminado sin posibilidad de recuperación.

## Qué debe incluir una copia de seguridad de WordPress

1. **Base de datos** — contiene todas tus publicaciones, páginas, ajustes, usuarios y comentarios
2. **Archivos** — contiene tu instalación de WordPress, temas, plugins y archivos multimedia subidos

Una copia de seguridad que incluye solo la base de datos o solo los archivos no es una copia de seguridad completa.

## La regla de las copias de seguridad 3-2-1

- **3** copias de tus datos
- **2** medios de almacenamiento diferentes
- **1** copia almacenada fuera del sitio (no en el mismo servidor que tu web)

## Los mejores plugins de copia de seguridad para WordPress

### UpdraftPlus (Gratuito + Premium) — El más popular

**Configuración:**
1. Instala y activa UpdraftPlus
2. Ajustes → UpdraftPlus Backups → pestaña Ajustes
3. Establece el programa: Diario para base de datos, Semanal para archivos
4. Elige almacenamiento remoto: Google Drive, Dropbox, Amazon S3, FTP
5. Haz clic en "Respaldar ahora" para probar inmediatamente

### BlogVault — El mejor para tranquilidad

BlogVault almacena copias de seguridad en sus propios servidores (independiente de tu host) y proporciona restauración con un clic. De pago desde 7,4 €/mes — vale la pena para webs críticas de negocio.

### Duplicator — El mejor para migraciones

Duplicator crea un paquete de sitio completo ideal tanto para copias de seguridad como para mover tu sitio a un nuevo host.

## Frecuencia de copias de seguridad recomendada

| Tipo de sitio | Base de datos | Archivos |
|---|---|---|
| Blog (posts diarios) | Diaria | Semanal |
| E-commerce (pedidos diarios) | Varias veces al día | Diaria |
| Web de presentación (actualizaciones raras) | Semanal | Mensual |
| En desarrollo activo | Antes de cada cambio | Antes de cada cambio |

## Probar tus copias de seguridad

Al menos dos veces al año: descarga una copia de seguridad a tu ordenador local, restáurala en un entorno de pruebas y verifica que todo funciona.
