# Core Web Vitals y Google AI Overview: todo lo que necesitas saber

El rendimiento de tu sitio web ya no es solo una cuestión técnica — es un factor de posicionamiento directo y un determinante clave de si tu contenido aparece en los resúmenes generados por IA. En 2026, entender los Core Web Vitals es imprescindible para cualquier negocio serio con la visibilidad orgánica.

## ¿Qué son los Core Web Vitals?

Los Core Web Vitals son tres métricas que Google utiliza para medir la experiencia real del usuario en tu sitio web.

### LCP — Largest Contentful Paint

Mide la rapidez con que carga el elemento visible más grande de tu página (normalmente una imagen hero o un encabezado).

- ✅ **Bueno:** menos de 2,5 segundos
- ⚠️ **Necesita mejora:** 2,5-4 segundos
- ❌ **Malo:** más de 4 segundos

**Cómo mejorar:** Optimiza y comprime imágenes, usa un CDN, elimina recursos que bloqueen el renderizado.

### INP — Interaction to Next Paint

Mide la rapidez con que tu página responde a las interacciones del usuario (clics, pulsaciones).

- ✅ **Bueno:** menos de 200ms
- ⚠️ **Necesita mejora:** 200-500ms

**Cómo mejorar:** Reduce el tiempo de ejecución de JavaScript, elimina scripts pesados de terceros.

### CLS — Cumulative Layout Shift

Mide la estabilidad visual — cuánto se desplaza inesperadamente el diseño de la página mientras carga.

- ✅ **Bueno:** menos de 0,1
- ⚠️ **Necesita mejora:** 0,1-0,25

**Cómo mejorar:** Especifica siempre dimensiones para imágenes y vídeos, reserva espacio para anuncios.

## Cómo medir tus Core Web Vitals

- **Google Search Console** → Informe de Core Web Vitals (datos de campo de usuarios reales)
- **PageSpeed Insights** (pagespeed.web.dev)
- **Chrome DevTools** → pestaña Lighthouse

Prioriza siempre los **datos de campo** sobre los datos de laboratorio.

## GEO: Core Web Vitals y la búsqueda con IA

Google AI Overviews, ChatGPT Search y Perplexity no solo extraen del contenido más relevante — favorecen cada vez más el contenido de sitios web técnicamente sólidos y de carga rápida.

**Las páginas rápidas se rastrean más profundamente.** Los sistemas de IA dependen de los datos de rastreo de Google. Las páginas lentas se rastrean con menos frecuencia.

**Checklist GEO para Core Web Vitals:**
- Alcanza puntuaciones "Buenas" en las tres métricas
- Añade datos estructurados FAQ para responder preguntas comunes directamente
- Usa encabezados H2/H3 claros que coincidan con consultas en lenguaje natural
- Mantén los párrafos cortos y las respuestas directas

## Mejoras rápidas para mejorar tus puntuaciones hoy

1. **Instala un plugin de caché** (WP Rocket, LiteSpeed Cache)
2. **Sirve imágenes en formato WebP** — 30% más pequeñas que JPEG/PNG
3. **Usa carga diferida** para imágenes por debajo del pliegue
4. **Elimina plugins y scripts sin usar**
5. **Cambia a un alojamiento de alto rendimiento**
