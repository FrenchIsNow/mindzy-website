# Refonte Menu Navigation - Site Mindzy

## Contexte
Étudie le design et la structure du site https://www.aleo.agency/ comme référence visuelle et tonale.

## Tâches

### 1. Mettre à jour la navigation principale
Supprimer tous les emojis du menu. Structure finale :
```
Solutions (dropdown) | Portfolio | Pourquoi nous | Blog | FAQ
```

### 2. Remplacer le header "Product" par un dropdown "Nos Services"
Chaque item du dropdown contient : icône + titre + sous-titre.
Chaque item redirige vers sa propre page dédiée.

| Titre | Sous-titre |
|-------|-----------|
| Site web | Une vitrine digitale incontournable |
| Solution sur mesure | Apps internes, mobile & agents IA |
| Formations & Réseaux Sociaux | Boostez votre notoriété |
| Logo & supports imprimés | Un univers à votre image |

### 3. Créer les pages correspondantes
Chaque entrée du dropdown doit avoir sa propre route/page (même si le contenu est placeholder pour l'instant).

### 4. Style & Ton
- S'inspirer du design épuré de https://www.aleo.agency/
- Textes courts, directs, professionnels
- Pas d'emojis nulle part dans le menu/header
- Réécrire les textes si nécessaire pour plus de concision

## Contraintes
- Garder la cohérence avec le design existant
- Navigation responsive (mobile hamburger menu inclus)
- Dropdown fluide avec animation subtile
